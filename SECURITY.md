# Security Policy

## Supported versions

Security fixes are handled on a best-effort basis for the latest code on the default branch.

| Version | Supported |
| --- | --- |
| `main` | Yes |
| Older snapshots and forks | No guarantee |

## Reporting a vulnerability

Please report suspected vulnerabilities privately by email:

- [sketchwarepromod@gmail.com](mailto:sketchwarepromod@gmail.com)

When possible, include:

- A short description of the issue
- Impact and affected area
- Reproduction steps or proof of concept
- Suggested mitigation, if you have one

Please do not open public GitHub issues for security vulnerabilities. Public reports can put users and contributors at risk before a fix is available.

## Response expectations

This is a community-maintained project, so response times can vary. We will review reports on a best-effort basis and may ask follow-up questions if more context is needed.
